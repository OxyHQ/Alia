# 🤖 Agentes de Alia

Alia es una plataforma multi-agente de IA donde cada agente está especializado en un dominio específico para brindarte la mejor asistencia posible.

## 🎯 Agentes Disponibles

### 1. **Alia** - Asistente General
El agente principal y más versátil de la plataforma.

**Especialidades:**
- Conversaciones generales
- Ayuda con tareas cotidianas
- Respuestas rápidas a preguntas variadas
- Coordinación con otros agentes especializados

**Cuándo usar:**
- Cuando no estés seguro de qué agente usar
- Para tareas generales y conversaciones
- Como punto de entrada a la plataforma

---

### 2. **Alia Developer** - Experto en Desarrollo
Especializado en arquitectura de software, debugging y desarrollo.

**Especialidades:**
- Arquitectura de software y patrones de diseño
- Debugging y resolución de errores
- Code reviews y optimizaciones
- Mejores prácticas de programación
- Múltiples lenguajes: JavaScript, TypeScript, Python, Go, Rust, Java, etc.
- Frameworks modernos: React, Next.js, Express, FastAPI, etc.
- DevOps y CI/CD

**Cuándo usar:**
- Cuando necesites ayuda con código
- Para diseñar arquitecturas de software
- Debugging y resolución de problemas técnicos
- Optimización de rendimiento
- Code reviews

**Ejemplos:**
```
"Ayúdame a diseñar la arquitectura de una aplicación de chat en tiempo real"
"Tengo un error en mi código React, ¿puedes ayudarme?"
"¿Cuál es la mejor manera de implementar autenticación JWT?"
"Revisa este código y sugiere mejoras"
```

## 🔄 Cambiar entre Agentes

En la interfaz de Alia, puedes cambiar fácilmente entre agentes según tus necesidades:

1. **En la App/Web**: Usa el selector de agentes en la parte superior
2. **En la API**: Especifica el agente en el parámetro `agent` o `model`

## 💡 Mejores Prácticas

### Para obtener mejores resultados:

1. **Sé específico**: Proporciona contexto claro sobre lo que necesitas
2. **Usa el agente correcto**: Cada agente está optimizado para su dominio
3. **Itera**: No dudes en refinar tus preguntas basándote en las respuestas
4. **Combina agentes**: Puedes usar varios agentes en una misma sesión

### Ejemplos de flujos de trabajo:

**Caso 1: Lanzamiento de Producto**
1. **Alia Business** → Análisis de mercado y estrategia
2. **Alia Developer** → Desarrollo del producto
3. **Alia Social Manager** → Estrategia de lanzamiento en redes

**Caso 2: Solución de Problema Técnico**
1. **Alia** → Descripción del problema
2. **Alia Developer** → Análisis técnico profundo y solución

**Caso 3: Campaña de Marketing**
1. **Alia Business** → Definir objetivos y métricas
2. **Alia Social Manager** → Crear contenido y estrategia
3. **Alia Business** → Analizar resultados y optimizar

- El tipo de tarea
- Disponibilidad y costos
- Preferencias del usuario
